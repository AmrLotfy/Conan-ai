القدرات

محقق بمهارات لا تحصى

CONAN لا يقترح — ينفّذ. من البحث في الويب إلى قراءة الملفات، القضية تُحسم دائماً.

🧠
ذاكرة المحقق
يتذكر كل تفصيلة ويتطور معك. لا قضية تُنسى — سياقك محفوظ عبر كل الجلسات.
MEMORY ENGINE
🔎
بحث حقيقي في الويب
لا تخمين، لا بيانات قديمة. يبحث في الإنترنت ويعود بنتائج حقيقية ومصادر موثّقة.
WEB SEARCH
📁
قارئ الملفات المحلية
اسأله "لخّص هذا الـ log" أو "ما الخطأ في هذا الملف" — يقرأ مباشرةً من جهازك.
FILE READER
🔧
نظام مهارات قابل للتوسع
أنشئ أدواتك كحزم npm. يُثبّتها المجتمع ويستخدمها الجميع فوراً.
SKILL ENGINE
💬
متعدد المزودين
استخدم مفتاحك من OpenAI أو Anthropic أو OpenRouter — اختيارك بالكامل.
MULTI-PROVIDER
🔒
خصوصية محلية كاملة
يعمل على جهازك. SQLite محلي. لا سيرفر خارجي، لا اشتراك، بياناتك ملكك.
